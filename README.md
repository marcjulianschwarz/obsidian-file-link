# Better File Link for Obsidian

![Better File Link Banner](https://user-images.githubusercontent.com/67844154/158657066-47b6b0fb-439c-4973-82c7-9768ee472344.png)

An Obsidian plugin for adding external file links to your notes. No more manual drag-and-dropping files. Instead, select files directly within Obsidian's interface.

## Key Features

- Choose files without leaving Obsidian
- Add custom prefixes for multiple file links
- Toggle file extensions visibility
- Choose between opening files or their containing folders
- Embed files directly in your notes
- Detect broken file links across all your notes (command: "Check for broken file links")

## Supported File Types for Embedding

- **Documents**: MD, PDF
- **Images**: PNG, JPG, JPEG, GIF, BMP, SVG
- **Audio**: MP3, WebM, WAV, M4A, OGG, 3GP, FLAC
- **Video**: MP4, WebM, OGV

## 🚀 Quick Start

1. Press `Cmd/Ctrl + P` to open the command palette
2. Search for "Add file link"
3. Click "Select files"
4. Choose your desired files
5. Toggle embedding option if needed
6. Click "Add file link"
